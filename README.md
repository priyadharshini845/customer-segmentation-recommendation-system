# customer-segmentation-recommendation-system

This project develops a Streamlit-based retail analytics dashboard that enables businesses to analyze customer purchasing behaviors and optimize marketing strategies. The dashboard integrates transactional data from a UK-based retailer, processing it to derive key customer insights such as sales trends, RFM (Recency, Frequency, and Monetary) values, and customer segmentation. K-Means, DBSCAN, and Agglomerative clustering algorithms are applied to segment customers based on their purchasing behavior, helping businesses target different customer groups with tailored promotions. Interactive visualizations using Plotly and Seaborn allow users to explore customer patterns, making data-driven decision-making more accessible and actionable.
To enhance customer engagement, the project implements a recommendation system that provides personalized product suggestions based on customer segmentation. By analyzing intra-cluster purchasing trends, the system recommends best-selling products within each segment, facilitating effective cross-selling and upselling strategies. Additionally, collaborative and content-based filtering techniques are integrated to refine recommendations based on individual purchase histories. The dashboard also includes advanced analytical features such as customer churn prediction using classification models, market basket analysis with association rule mining , and automated report generation to assist businesses in improving retention and revenue growth.
